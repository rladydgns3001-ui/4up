워드프레스 사이트 몇 개까지 연동할 수 있나요? +
Basic, Pro 모두 워드프레스 연동 수에 제한이 없습니다. 사이트 1개든, 10개든 동일하게 사용 가능합니다. 여러 블로그를 동시에 운영하고 싶은 분께 특히 유리합니다.
구매 후 어떻게 받나요? 파일을 어디서 다운받나요? +
결제 완료 즉시 입력하신 이메일로 다운로드 링크가 전송됩니다. 이후 카카오톡으로 연락 주시면 1:1 줌 설치 교육 일정을 잡아드립니다. 설치에 막히는 부분은 교육 시간에 화면 공유로 함께 해결합니다.
소스코드도 함께 제공되나요? +
소스코드는 별도로 제공되지 않습니다. 실행 파일(exe) 형태로 제공되며, 소스코드 없이도 모든 기능을 완전하게 사용할 수 있습니다. 추가 커스터마이징이 필요하신 경우 별도로 문의해 주세요.
글이 저품질 판정을 받지는 않나요? +
AutoPost는 단순 복붙이 아니라 키워드 기반 구조화된 SEO 글을 생성합니다. 메타태그, H태그, 내부 링크, 적절한 키워드 밀도까지 자동으로 최적화되어 구글 저품질 위험을 최소화합니다. 실제 사용자 98%가 만족하고 있습니다.
ChatGPT API가 뭔가요? 추가 비용이 드나요? +
AutoPost는 ChatGPT(OpenAI) API를 사용해 글을 생성합니다. API는 openai.com에서 별도로 신청하며, 사용량에 따라 비용이 발생합니다. 글 1개당 약 20~50원 내외로 매우 저렴합니다. 설치 교육 시 API 발급 방법도 함께 안내해 드립니다.
Mac에서도 사용할 수 있나요? +
현재 AutoPost는 Windows 전용입니다. Mac 사용자는 가상머신(Parallels, VMware) 등을 통해 Windows 환경에서 사용할 수 있습니다. Mac 네이티브 버전은 추후 업데이트 예정입니다.
Basic과 Pro, 뭘 사야 할지 모르겠어요 +
처음 시작하거나 천천히 운영하고 싶다면 Basic, 하루 여러 개 발행하고 구글 색인을 빠르게 원하면 V2 Pro를 추천합니다. 실제로 구매자의 80%가 Pro를 선택합니다. 결정이 어려우시면 카카오톡으로 운영 목표를 말씀해 주시면 딱 맞는 플랜을 추천해 드립니다.
환불은 되나요? +
AutoPost는 디지털 프로그램 특성상 구매 후 환불이 불가합니다. 구매 전 반드시 카카오톡 상담을 통해 궁금한 점을 모두 확인하신 후 결제해 주세요. 구매 전 충분한 설명과 데모 영상을 제공하고 있으니 부담 없이 문의해 주세요. 자세한 내용은 환불 규정을 확인해 주세요.

FAQ

자주 묻는 질문

워드프레스는 최대 3개 사이트까지 연동 가능합니다. 네이버 블로그는 저품질 방지를 위해 IP를 고정하여 1개 계정만 사용하는 것을 권장합니다.
단순 복붙이 아닌 키워드 기반 구조화된 SEO 글을 생성합니다. 메타태그, H태그, 키워드 밀도까지 자동으로 최적화되어 구글 저품질 위험을 최소화합니다. 실제 사용자 98%가 만족합니다.
AutoPost는 Claude API를 사용해 글을 생성합니다. 글 1개당 약 20~50원 내외로 매우 저렴합니다. API 발급 방법은 설치 교육 시 안내해 드립니다.
Windows와 Mac 모두 사용 가능합니다.
카카오톡으로 결제 안내 → 이메일로 프로그램 파일 및 가이드 전송 → 1:1 줌 설치 교육 (30분) → 구매 후 세팅하면 바로 발행 가능합니다.
디지털 프로그램 특성상 구매 후 환불이 불가합니다. 구매 전 카카오톡으로 충분히 상담 후 결제해 주세요.
워드프레스 블로그만 운영하신다면 V2 Pro, 네이버 블로그 자동화가 필요하시면 AutoPost NB를 추천합니다. 워드프레스와 네이버 둘 다 자동화하고 외부유입까지 원하시면 올인원 패키지가 가장 합리적입니다.

환불 규정

최종 수정일: 2026년 2월 15일

AutoPost SEO Writer(이하 "서비스")의 환불 규정을 안내드립니다. 구매 전 아래 내용을 반드시 확인하시기 바랍니다.

1. 상품 전달 방식

AutoPost SEO Writer는 디지털 소프트웨어 상품입니다. 결제 완료 즉시, 결제 시 입력하신 이메일 주소로 프로그램 파일 및 사용 가이드가 자동 발송됩니다.

  • 별도의 이메일 주소를 수집하지 않으며, Polar 결제 화면에서 입력하신 이메일로만 발송됩니다.
  • 이메일 발송은 결제 완료 후 자동으로 즉시 처리되며, 수동 개입 없이 시스템에서 자동 진행됩니다.

2. 이메일 주소 오류에 대한 책임

결제 시 입력하신 이메일 주소가 정확해야 합니다.

이메일 주소를 잘못 입력하여 프로그램을 수신하지 못한 경우, 환불이 불가합니다.

  • 결제 전 이메일 주소를 반드시 확인하시기 바랍니다.
  • 오타, 존재하지 않는 이메일, 수신 불가 이메일 등 구매자의 부주의로 인한 전달 실패는 판매자의 책임이 아닙니다.
  • 이메일이 시스템에서 정상적으로 발송된 것이 확인되는 경우, 수신 여부와 관계없이 전달 완료로 간주합니다.

3. 자동 환불 (시스템 오류 시)

이메일 발송이 시스템 오류로 실패한 경우, 전액 자동 환불됩니다.

  • 프로그램 전달 이메일이 서버 오류, API 장애 등 판매자 측 원인으로 발송 자체가 실패한 경우, 결제 금액이 자동으로 즉시 전액 환불 처리됩니다.
  • 자동 환불은 시스템에서 별도의 요청 없이 자동으로 진행됩니다.
  • 환불은 결제 대행사(Polar)를 통해 원래 결제 수단으로 반환됩니다.

4. 환불 불가 (전달 완료 후)

이메일 발송이 완료된 이후에는 어떠한 사유로도 환불이 불가합니다.

  • 프로그램 특성상 이메일로 발송이 완료되면 프로그램에 대한 모든 권한이 구매자에게 이전됩니다.
  • 디지털 상품의 특성상 발송된 프로그램을 회수하는 것이 기술적으로 불가능하므로, 전달 완료 후에는 환불을 처리할 수 없습니다.
  • 이는 단순 변심, 사용 의향 변경, 기대와 다른 기능 등 모든 사유에 해당합니다.

5. 환불 가능한 예외 경우

위 원칙에도 불구하고, 아래의 경우에 한하여 환불을 처리해드립니다:

  • 중복 결제가 발생한 경우 — 확인 후 즉시 환불 처리
  • 소프트웨어의 중대한 결함으로 정상 작동이 불가능하고, 판매자의 기술 지원으로도 해결이 되지 않는 경우

6. 환불이 불가능한 경우 (요약)

  • 이메일 발송이 정상적으로 완료된 후 단순 변심에 의한 환불 요청
  • 결제 시 이메일 주소를 잘못 입력하여 프로그램을 수신하지 못한 경우
  • 이용자의 운영 환경(PC 사양, 네트워크, OS 호환성 등)에 의한 문제
  • 외부 API(Claude API, OpenAI API 등)의 정책 변경이나 비용 변동으로 인한 환불 요청
  • 서비스 약관을 위반하여 이용이 제한된 경우

7. 환불 절차

  1. 자동 환불: 시스템 오류로 이메일 발송이 실패한 경우, 별도 요청 없이 자동으로 전액 환불됩니다.
  2. 수동 환불 요청 (예외 사항): 카카오톡 오픈채팅으로 환불 사유와 결제 정보를 알려주세요.
  3. 검토: 환불 요청 접수 후 영업일 기준 3일 이내에 검토 결과를 안내드립니다.
  4. 환불 처리: 환불이 승인되면 결제 대행사(Polar)를 통해 환불이 진행됩니다. 결제 수단에 따라 실제 환급까지 영업일 기준 5~10일이 소요될 수 있습니다.

8. 결제 대행사

본 서비스의 결제는 Polar(polar.sh)를 통해 처리됩니다. Polar는 Merchant of Record(MoR)로서 결제 처리, 세금 징수, 환불 처리를 담당합니다. 결제 관련 영수증은 Polar에서 자동으로 발급됩니다.

9. 구매 전 확인 권장사항

환불이 제한적이므로, 구매 전 아래 사항을 충분히 확인하시기를 권장합니다:

  • 결제 시 이메일 주소를 정확하게 입력하세요. 잘못된 이메일로 인한 환불은 불가합니다.
  • 상품 상세 페이지에서 기능을 확인하세요.
  • 데모 영상을 통해 실제 동작을 확인하세요.
  • 카카오톡 오픈채팅으로 사전 상담을 받으세요.
  • 운영 환경(Windows 또는 macOS)이 적합한지 확인하세요.

10. 관련 법령

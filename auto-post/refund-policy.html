환불 규정

최종 수정일: 2026년 2월 15일

AutoPost SEO Writer(이하 "서비스")의 환불 규정을 안내드립니다. 구매 전 아래 내용을 반드시 확인하시기 바랍니다.

1. 기본 환불 정책

AutoPost SEO Writer는 디지털 소프트웨어 상품으로, 결제 완료 후 즉시 다운로드 링크가 제공됩니다. 디지털 상품의 특성상 다운로드 또는 설치가 완료된 후에는 원칙적으로 환불이 어렵습니다.

2. 환불 가능한 경우

아래의 경우에 한하여 환불을 처리해드립니다:

  • 결제 후 소프트웨어를 다운로드하지 않은 경우 — 결제 후 7일 이내 요청 시 전액 환불
  • 중복 결제가 발생한 경우 — 확인 후 즉시 환불 처리
  • 소프트웨어의 중대한 결함으로 정상 작동이 불가능하고, 판매자의 기술 지원으로도 해결이 되지 않는 경우

3. 환불이 불가능한 경우

  • 소프트웨어를 다운로드하거나 설치를 완료한 후 단순 변심에 의한 환불 요청
  • 이용자의 운영 환경(PC 사양, 네트워크, OS 호환성 등)에 의한 문제
  • 외부 API(Claude API, OpenAI API 등)의 정책 변경이나 비용 변동으로 인한 환불 요청
  • 서비스 약관을 위반하여 이용이 제한된 경우

4. 환불 절차

  1. 환불 요청: 카카오톡 오픈채팅으로 환불 사유와 결제 정보를 알려주세요.
  2. 검토: 환불 요청 접수 후 영업일 기준 3일 이내에 검토 결과를 안내드립니다.
  3. 환불 처리: 환불이 승인되면 결제 대행사(Polar)를 통해 환불이 진행됩니다. 환불 처리 후 결제 수단에 따라 실제 환급까지 영업일 기준 5~10일이 소요될 수 있습니다.

5. 결제 대행사

본 서비스의 결제는 Polar(polar.sh)를 통해 처리됩니다. Polar는 Merchant of Record(MoR)로서 결제 처리, 세금 징수, 환불 처리를 담당합니다. 결제 관련 영수증은 Polar에서 자동으로 발급됩니다.

6. 구매 전 확인 권장사항

환불이 제한적이므로, 구매 전 아래 사항을 충분히 확인하시기를 권장합니다:

  • 상품 상세 페이지에서 기능을 확인하세요.
  • 데모 영상을 통해 실제 동작을 확인하세요.
  • 카카오톡 오픈채팅으로 사전 상담을 받으세요.
  • 운영 환경(Windows 또는 macOS)이 적합한지 확인하세요.

7. 관련 법령

본 환불 규정은 「전자상거래 등에서의 소비자보호에 관한 법률」 및 관련 법령에 따릅니다. 디지털 콘텐츠의 경우 「콘텐츠산업 진흥법」에 의거하여 다운로드 또는 사용이 개시된 이후에는 청약철회가 제한될 수 있습니다.
